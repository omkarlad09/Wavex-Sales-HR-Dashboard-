🚀 Wavex Sales & HR Analytics Dashboard

📊 **End-to-End Business Intelligence Project using Power BI**

🔍 Project Overview

This project focuses on analyzing **Sales and HR data of Wavex company** to generate actionable insights and support data-driven decision making.

The dashboard provides a **360° view of business performance**, covering revenue trends, employee analytics, and operational efficiency.

🎯 Objectives

* Analyze overall **sales performance & revenue trends**
* Identify **top-performing products & regions**
* Understand **employee distribution & attrition**
* Track **department-wise performance**
* Enable better **business decision-making**

📊 Key Insights

✔️ Monthly & yearly **sales trends**
✔️ Top products contributing to revenue
✔️ Region-wise sales performance
✔️ Employee count by department
✔️ Attrition rate & workforce insights

🛠️ Tools & Technologies

* 📌 Power BI (Data Visualization)
* 📌 Excel / CSV (Data Source)
* 📌 DAX (Data Analysis Expressions)
